API Call To Welsh Text to Speech Service Intermediate Javascript Project

The final project in my Intermediate Javascript class included researching and using API calls. I was working on a site of Welsh language vocabulary games. The University of Wales at Bangor has an amazing set of resources for developing Welsh websites at Tech Iaith. This page has a simple form to enter a word or sentence in Welsh. After sending the request, the site loads a wav file of an AI voice pronouncing it in Welsh.

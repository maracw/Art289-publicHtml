Penguin Valentine Mystery Game Baby's first Javascript

This came out a class for folks completely new to programming. We used Code.Org's Javascript curriculum along with lectures and work sessions. Code.org's visual interface really worked for me. I still think of variables as purple, and loops as blue.

This penguin game is based on a computer program printed in Ranger Rick magazine sometime in the late 1980s. I read the code out line by line and my Dad typed the commands into the DOS machine he brought home from work. The plot of the program involved Pandas, zookeepers, and deeds most foul. I adapted the story to be about penguins, rather than poison.

The penguin names and properties are random each time you play the game. So Rosie may not be the penguin you are looking for..
